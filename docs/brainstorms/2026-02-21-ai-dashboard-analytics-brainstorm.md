# AI Dashboard Analytics

## What We're Building
Добавляем на дашборд отдельный блок AI-аналитики, который на основе текущих данных (продажи, YouTube, Telegram, календарь) автоматически показывает:
- интегральный AI Health Score,
- краткую интерпретацию ситуации,
- ключевые сигналы (рост/падение, концентрация, нагрузка),
- 3 горизонта действий: 24ч, 7д, 30д,
- рекомендации с приоритетом и ожидаемым эффектом.

## Why This Approach
Выбран lightweight-подход без внешней LLM: deterministic-аналитика по уже доступным данным API. Расчёты перенесены в backend (`/api/dashboard`) для единой логики на всех клиентах.

## Key Decisions
- AI-логика считается в backend и отдаётся в `dashboard.ai`.
- Используем простые объяснимые метрики: WoW-тренды, волатильность, концентрация топ-товара, загрузка календаря.
- Результат actionable: оценка + сигналы + рекомендации (priority/effect).
- Структура UI: общий AI-блок сверху + 3 компактные карточки горизонтов.
- KPI-веса: бизнес-эффект 60, скорость решений 20, качество решений 20.

## Open Questions
- Нужна ли в следующей итерации генерация текстового summary через внешнюю LLM?
- Нужна ли ручная настройка порогов для priority/effect по рекомендациям?
